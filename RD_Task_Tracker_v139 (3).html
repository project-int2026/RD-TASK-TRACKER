🖥️
ข้อมูลอยู่บนเครื่องของคุณเท่านั้น
ข้อมูลทุกอย่างถูกเก็บใน localStorage ของ Browser บนเครื่องนี้เท่านั้น ไม่มีการส่งข้อมูลออกไปยัง Server ใดๆ ทั้งสิ้น
🚫
ไม่มี Cloud — ไม่มี Server
โปรแกรมนี้ทำงานแบบ Offline ได้ 100% ไม่มีการเชื่อมต่อฐานข้อมูลภายนอก ไม่มี API Call สำหรับข้อมูล Task ของคุณ แม้แต่ claude.ai เองก็ไม่สามารถเข้าถึงข้อมูลได้
💾
Backup ข้อมูลด้วยตัวเอง
ใช้ปุ่ม 💾 BACKUP บน Header เพื่อดาวน์โหลด JSON file สำรองข้อมูลไว้บนเครื่อง และนำกลับมา Restore ได้ทุกเมื่อ
⚠️
หมายเหตุสำคัญ
หากล้าง Cache / Clear Browser Data ข้อมูลใน localStorage จะหายไป กรุณา Backup ข้อมูลสม่ำเสมอ
